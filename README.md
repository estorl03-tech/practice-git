
# Git・GitHub 基礎習得プロジェクト

このリポジトリは、チーム開発で必須となる Git の基本操作と、現場で発生する課題への対応力を養うために作成しました。

## 習得したスキル
- **ブランチ運用**: `main` から作業用ブランチを切り、安全に開発を進めるフローの習得。
- **プルリクエスト (Pull Request)**: GitHub上でのコード差分確認と、マージプロセスの実践。
- **競合（Conflict）の解消**: 
  - ブラウザとローカルでの同時編集による競合をあえて発生させ、マージエディターを用いて解決。
  - 「現在の変更」と「着信側の変更」を比較し、適切なコードを選択・合流させる手順を習得。


## 使用ツール
- Git / GitHub
